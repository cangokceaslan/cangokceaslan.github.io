INSTALLATION

Follow the steps below to setup your site template:

Unzip the downloaded package and open the e&e folder to find all the template files. You will need to upload these files to your hosting web server using FTP in order to use it on your website.

You should upload all or specific HTML files as per your need. You're now good to go..! Start adding your content/images and generate your beautiful brand new website for your awesome shop.

JavaScript

This template imports seven Javascript files.

  1. jquery-1.12.0.min.js
  2. modernizr-2.8.3.min.js ( called in head )
  3. popper.min.js
  4. bootstrap.min.js
  5. plugins.js ( contain all third party plugins )
  6. main.js ( contain all plugin activation )

 

Customization

Favicon Seatings

To change the favicon image of your site, you need to load a new ICO image in assets/images Folder or you change the path as you need.

<link rel="shortcut icon" type="image/x-icon" href="assets/images/favicon.ico">

 

Font Changing

We use google font for E&E. To change them go to google font select font you want to use and get import code and replace theme in style.css file.

@import url('https://fonts.googleapis.com/css?family=Open+Sans:300,400,600,700,800|Oswald:200,300,400,500,600,700');
.demo-class {
    font-family: 'Open Sans', sans-serif;
}

 

Sticky Header Seatings

We use header as sticky but if you want to change sticky header to normal header go to header-bottom and remove header-sticky class.

<div class="header-bottom header-bottom-one header-sticky">

 

Logo Changing

To change  logo go to header-bottom and fond header-logo element and change the image path with your logo path.

<div class="header-logo">
    <a href="index.html">
        <img src="assets/images/logo.png" alt="E&E - Electronics Ecommerce Bootstrap4 HTML Template">
        <img class="theme-dark" src="assets/images/logo-light.png" alt="E&E - Electronics Ecommerce Bootstrap4 HTML Template">
    </a>
</div>

Note: We use 2 logo. logo.png for ligth version & logo-light.png for dark version. Call both logo it will change depending on whitch version you are using.

 

 

Footer Twitter Feed Settings

To show your twitter feed in footer widget you have to edit config.php file.

File path: e&e/assets/api/config.php

<?php
    /**
     * Your Twitter App Info
     */
    
    // Consumer Key
    define('CONSUMER_KEY', 'CONSUMER KEY HERE');
    define('CONSUMER_SECRET', 'CONSUMER SECRET HERE');

    // User Access Token
    define('ACCESS_TOKEN', 'ACCESS TOKEN HERE');
    define('ACCESS_SECRET', 'ACCESS SECRET HERE');
    
    // Cache Settings
    define('CACHE_ENABLED', false);
    define('CACHE_LIFETIME', 3600); // in seconds
    define('HASH_SALT', md5(dirname(__FILE__)));
